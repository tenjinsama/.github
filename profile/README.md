## Description

Welcome to pod pda, a learning project about creating a PDA (Personnal Digital Assistant) device for any geek.  

### Why This project ?

- To learn about iot, rapsberry pi, embedded system.
- To have controller over the device (customize at will).

### Projects components :

This project is sperated in 2 components "backend" (back) and "embedded" (emb). To learn more about these 2 components, look at parts below.

- [**Pod Backend**](https://github.com/pod-pda/pod-back) : The backend system that work with the "pod".
- [**Pod embedded**](https://github.com/pod-pda/pod-emb) : The embedded system (the "pod") based on raspberry pi.
